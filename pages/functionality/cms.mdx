# Content Management System

Content Management System (CMS) is a feature that allows you to manage the dynamic content on Webflow. And this is where the Goat Slider shines. It allows you to create sliders with dynamic content based on your latest blogs or different filtering methods that can be done using Webflow.

<Callout type='info'>
    Native Webflow Slider doesn't support this feature yet!
</Callout>
